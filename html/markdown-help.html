Gruber’s Markdown

Emphasis

Italic - italic

*italic*

Bold - bold

**bold** or __bold__

Links

Inline - link

A [Link](http://example.com "Title")

Reference-style - [link][id]

A [Link][id].

[id]: http://example.com "Title"

Code

A code span

A `code span`

Image

Github Flavored Markdown

Strikethrough - ~strikethrough~

Line Breaks

End a line with tow or more spaces

Roses are red,
Violets are blue.

Roses are red,  
Violets are blue.

